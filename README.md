# Runner

Runner is a service that can be talked to via IPC in order to retrieve lists of items which can be used to f.e. build an application launcher.

## Planned features

- extendable via plugins
- history
- fuzzy search

## Planned providers

- desktop applications
- windows
- binaries
- calculator
- websearch
- clipboard
- custom commands
- finder
- emojis
- dmenu
